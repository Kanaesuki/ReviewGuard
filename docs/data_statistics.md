# 数据集统计报告

## 1. 数据来源与标签定义

- 数据来源：京东电商真实/虚假评论数据集
- 标签 0：虚假/水军评论
- 标签 1：真实用户评论

## 2. 数据规模

| 数据集 | 样本数 |
| --- | ---: |
| train | 5556 |
| dev | 2000 |
| test | 2000 |
| 合计 | 9556 |

## 3. 类别分布

### 全量分布

| 类别 | 数量 | 占比 |
| --- | ---: | ---: |
| 虚假/水军评论 (0) | 5783 | 60.52% |
| 真实用户评论 (1) | 3773 | 39.48% |

### train

- 虚假/水军评论：3349 条（60.28%）
- 真实用户评论：2207 条（39.72%）

### dev

- 虚假/水军评论：1225 条（61.25%）
- 真实用户评论：775 条（38.75%）

### test

- 虚假/水军评论：1209 条（60.45%）
- 真实用户评论：791 条（39.55%）
