# 课堂展示提纲

## 1. 任务背景

- 虚假评论和水军评论会误导用户决策。
- 本项目将其抽象为文本二分类任务。

## 2. 系统流程

```text
评论数据 -> 文本清洗 -> 中文分词 -> TF-IDF/n-gram -> 分类模型 -> 识别结果
```

## 3. 实现过程

- 构造 `text,label` 数据集。
- 清洗噪声文本。
- 使用 jieba 分词。
- 使用 TF-IDF 提取文本特征。
- 训练并比较多个分类模型。

## 4. 技术挑战

- 中文分词和停用词处理。
- 虚假评论表达模板化但会伪装。
- 数据类别不平衡。
- 只看指标不够，需要案例解释。

## 5. 解决方案

- 使用 n-gram 捕获短语模式。
- 使用 class weight 缓解类别不平衡。
- 用混淆矩阵和错误案例分析解释模型效果。
- 做 Streamlit 页面展示真实输入预测过程。

## 6. 实验结果

- 展示模型对比表。
- 展示混淆矩阵。
- 展示典型案例：
  - 正确识别的水军评论。
  - 正确识别的正常评论。
  - 被误判的边界评论。

## 7. 演示环节

- 运行训练脚本。
- 打开 Streamlit 页面。
- 输入三类评论：
  - 明显正常评论。
  - 明显水军评论。
  - 模糊边界评论。

## 8. 总结与改进

- 当前系统已完成基础文本检测流程。
- 后续可加入用户行为特征、评论时间特征、深度学习模型和更大规模数据。
